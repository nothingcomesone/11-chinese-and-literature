A 第一部分：5 大修辞手法深度解析 Part A: Five Literary Techniques in Depth

📚 为什么要学这 5 大手法？Why These Five?

白先勇的《寂寞的十七岁》之所以是华文文学经典，不在于情节有多戏剧化，而在于它用 5 种文学手法层层叠加，把一个普通台北少年的崩溃精确地传递给读者。

这 5 大手法在 HSC 考试中的作用：

• Q3 (b) (5 分)：直接考「分析作者使用了什么手法 + 效果」——TEE 模型

• Q4 (25 分)：跨文本比较时，每一个论据段都要镶嵌一个「mini-TEE 技巧分析」——TEEL 模型

• Q6/Q7 (25 分)：一篇 5 段演讲稿至少要有 1-2 处用到这些手法（特别是排比 / 反问 / 对比）来增强语言的感染力

→ 掌握 5 大手法 = 同时为 Q3、Q4、Q6 三道题打地基。

A1 手法一：心理描写 Psychological Description

项目 内容
📖 定义 Definition 通过第一人称内心独白 + 生理反应（颤抖 / 心跳 / 冷汗 / 麻木）来外化人物的情绪状态。让读者不只是「看」人物，而是「成为」人物。
📜 文中例子 (原文引用)

• Scene 6 唐爱丽强吻：「我发觉我的心在发抖。」

• Scene 6：「我的脸烧得滚烫」「喉咙发干，快讲不出话」

• Scene 7 黑板信：「我没有出声，我发觉我全身在发抖」

• Scene 8 父亲耳光：「脸上痛得快淌眼水了，可是我拼命抵住，不让眼泪流下来。在爸爸面前，我不想哭。」
✨ 效果 Effect 把无形的「恐惧 / 屈辱 / 自我抑制」外化为读者可感知的身体颤抖。读者不是在「听故事」，而是在「与杨云峰共振」——这种「身体共振」是文学手法中最直击人心的。
🎯 与议题关联 对应议题 2.1 的「心理健康问题」与「个人 / 情感压力」——展示青少年心理崩溃在身体层面的具体形态。
✏️ 你的练习 (在 Scene 6/7/8 中再找 1 处)

A2 手法二：反复 Repetition

项目 内容
📖 定义 把同一类词语 / 句式 / 意象在不同场景反复出现，形成人物的「心理姿态」或「叙事母题」。反复不是啰嗦——它是「不断回到同一个深渊」的语言节奏。
📜 文中例子

• 「钻 / 躲 / 蒙」三个动作贯穿全篇：「钻到被窝里」/「躲进自己房里」/「蒙头大睡」/「我把被窝蒙住头，搂紧了枕头」（结尾）

• 「出家当和尚」的执念在故事中反复出现 4-5 次（如 Scene 7 后段「我又一个人在打空电话了。我告诉魏伯飏听，我真的想出家当和尚」）

• 父亲的反问句反复轰击：「怎么念得好？」「中什么用呀！」「想做什么呀？」「你还是个人哪？」
✨ 效果 反复让读者在节奏层面「记住」杨云峰的心理姿态——他不是某一刻崩溃的，他是「一直在缩进黑暗」。每一次「钻」「躲」「蒙」的反复，都把这种被动退缩变得更深、更难逃出。
🎯 与议题关联 对应议题 2.1「应对方式」中的「逃避 vs 有效应对」——杨云峰反复「钻 / 躲 / 蒙」展示了一种持续性的逃避型应对，是「躺平」的早期文学原型。
✏️ 你的练习

A3 手法三：对比 Contrast

项目 内容
📖 定义 把两个相反的画面 / 人物 / 情境并置，让对比本身说话——比直接评论更有力量。对比可以是「人物对比」「环境对比」「动静对比」「时空对比」。
📜 文中例子

• Scene 6 色彩对比：唐爱丽「西洋红的呢大衣」「鲜红的紧身毛衣」「涂口红」（外向 / 危险）vs 杨云峰「打了一寒战」「脸烧得滚烫」（拘谨 / 退缩）

• Scene 7 动静对比：同学「捧着肚子怪叫」「笑得前仰后翻」（群体狂欢）vs 杨云峰「全身在发抖」「没有出声」（个体僵死）

• Scene 8 时间对比：父亲动作「慢得叫人心焦」（拖延 / 压抑）vs 突然「举手一巴掌打在我脸上」（爆发 / 暴力）

• 全篇身份对比：杨云峰「肮脏 / 泥浆 / 血污」 vs 制服「浆得挺硬」「叠得整整齐齐」
✨ 效果 对比让读者在「两幅画面之间」感受到张力——不需要作者解释，对比本身就是论证。Scene 7 的群嘲与僵死对比，比直接说「他被霸凌」要锋利十倍。
🎯 与议题关联 对应议题 2.1「同伴 / 社交压力」与「家庭压力」——对比手法精准呈现了「集体的笑」与「个体的痛」、「社会的期待」与「真实的我」之间不可调和的撕裂。
✏️ 你的练习

A4 手法四：象征 Symbolism

项目 内容
📖 定义 用一个具体的物 / 场景 / 动作，承载一个抽象的情感或主题。象征不是直白的隐喻——它要求读者「悟」出其内涵。
📜 文中例子

• 黑板（Scene 7）—— 教室的中心权威被翻转：本应写知识的黑板，现在钉着杨云峰最脆弱的告白信。象征「私密被公开化」、「求助被武器化」

• 父亲耳光（Scene 8）—— 不只是身体暴力，更象征「家族颜面权威」对个体的暴击。两巴掌之后是「我们杨家没有这种人」——肉体伤害与身份否定一并降临

• 紫光路灯（Scene 9）—— 「路灯发着紫光，照在皮肤上，死人颜色一般」。象征杨云峰精神已死的预感

• 铁轨（Scene 9 末）—— 最强的死亡象征。「我真的趴到铁轨上去过，有一辆柴油快车差点压到我身上来。」铁轨不是「想自杀」的隐喻，而是「精神崩溃临界点」的具象化
✨ 效果 象征让小说获得「文学厚度」——读者读到「黑板上的信」时，看到的不只是一封信，而是「整个时代对青少年隐私的暴力」。象征让一个个体故事承载一个时代的集体困境。
🎯 与议题关联 对应议题 2.1 全部 6 大压力源——象征手法把抽象的「学业 / 家庭 / 同伴 / 心理」压力转化成具体的「黑板 / 耳光 / 铁轨」物象，这就是为什么这部小说能成为议题 2.1 的经典文本。
✏️ 你的练习

A5 手法五：反问 Rhetorical Question

项目 内容
📖 定义 形式上是问句，实质上不期待回答——答案藏在问句本身。反问可以是「攻击性反问」（剥夺对方人格）或「内心反问」（绝望的自我对话）。
📜 文中例子

• 父亲的攻击性反问（Scene 8）：「你还是个人哪？」「想做什么呀？」「怎么念得好？」——每一个反问都不是真问，而是用问句的形式剥夺杨云峰的人格

• 母亲的情感绑架反问：「还不是我命苦？」「中什么用呀！」——通过反问把自己的不幸归罪于儿子

• 杨云峰的内心反问：「你说叫我跟谁去说话？」——这是全篇最凄凉的一句反问，因为它没有人听。一个少年对着虚空发问，这就是「凝视的缺席」最深的形态

• Scene 8 后端杨云峰内心：「无论谁再要对我讲一句重话，我就发疯了。」——这其实是一句「省略形式的反问」（= 你们还要逼我到什么地步？）
✨ 效果 反问是「权力关系的语言形式」：施加反问的人（父亲 / 母亲）不期待回答，只期待沉默；承受反问的人（杨云峰）没有还击的语言空间。这种语言形式比直接辱骂更摧毁——因为它把「服从」伪装成「回答」。
🎯 与议题关联 对应议题 2.1「家庭压力」中的「父母期望 / 望子成龙 / 面子文化」——反问是中国传统家庭权威表达期待的典型语言形式。理解反问，就是理解「冷暴力」的语言层面。
✏️ 你的练习

我没有参加大考。我在日记本上写了几个大字：「杨云峰，你完蛋了！」

「哦，在公园里呢！你还告诉我考得不错——」

爸爸气得声音抖了，伸手又给了我一个巴掌。我脸上痛得快淌眼水了，可是我拼命抵住，不让眼泪流下来。在爸爸面前，我不想哭。

C 第三部分：TEE → TEEL 桥接训练 Part C: Bridging from TEE to TEEL

🌉 为什么需要「桥接」？The Bridge from TEE to TEEL

W2 我们学会了 TEE（Technique → Example → Effect, ~100 字, Q3 b 用）。今天 W3 我们要把它升级到 TEEL（Topic → Extend → Evidence → Link, ~350 字, Q4/Q6 段用）。

关键区别：TEE 围绕「一个技巧」展开；TEEL 围绕「一个论点」展开，论点的论据里可以镶嵌 mini-TEE 技巧分析。

今天的训练分三步：

① C1：在 Scene 6/7/8 上各写 1 个 TEE 微段（围绕一个手法）

② C2：用 3 个 TEE 拼出 1 个完整 TEEL 段（围绕一个论点）——这是 W2 没做透的关键能力

③ Part D：把 TEEL 段升级到完整的 5 段 TEEL 演讲稿（W3 的最终目标）

📜 TEEL 范例（参考，不要照抄）

T（Topic）：杨云峰之所以最终走到铁轨边，不是因为某一个具体打击，而是因为「心理压抑 / 同伴霸凌 / 家庭暴力」三重暴力以连续的频率叠加在一个十七岁少年身上——而这种「叠加性」正是当代「内卷青年」困境最古老的文学原型。

E（Extend）：单一的暴力是可承受的——一次责骂、一次失败、一次羞辱，都不足以击垮一个少年。但当三种暴力以每周、甚至每天的节奏叠加，一个本就缺乏「被看见」体验的少年就会逐渐丧失对自我存在的确认。这与 W2 学过的「凝视的缺席」是同一现象的两面：缺席让人看不见自己，叠加让人无处可逃。

E（Evidence + 镶嵌 mini-TEE）：在 Scene 6 唐爱丽暗教室一段，白先勇用心理描写——「我发觉我的心在发抖」「喉咙发干，快讲不出话」——把抽象的恐惧外化为读者可感的颤抖（mini-TEE：心理描写 / 生理反应外化 / 让读者与人物共振）。Scene 7 黑板信场景中，作者通过群嘲与僵死的强烈对比——同学「捧着肚子怪叫」与杨云峰「全身在发抖」「没有出声」——让校园霸凌不需任何评论便锋利显现（mini-TEE：动静对比 / 不评论自现）。而到了 Scene 8 父亲耳光，反问的破坏力达到顶点——「你还是个人哪？」「想做什么呀？」——这些反问不期待回答，只期待沉默；它们以语言的形式剥夺了一个十七岁少年的人格（mini-TEE：攻击性反问 / 权力关系语言化）。

L（Link）：这三种手法叠加的不是「三种修辞」，而是「三层暴力」——心理的、同伴的、家庭的。读懂这种叠加，就读懂了为什么《寂寞的十七岁》能成为议题 2.1 的文学原型：今天的「内卷青年」之所以累，不是任何单一压力造成的，而正是「学业 + 家庭 + 同伴 + 数字时代」多重压力以叠加方式同时降临——杨云峰六十年前的崩溃，是当代青年困境的最早预演。

D2 完整 5 段 TEEL 范文 Complete 5-Paragraph TEEL Speech

立场（一边倒）：是的，年轻人有第三条路 ——「主动设计」，区别于「被动反应」（内卷 = 被动加速 / 躺平 = 被动撤退）。

📜 范文 · 第 1 段（引言 ~95 字）

亲爱的毕业班同学们：

大家好！今天我想和大家聊一个我们这一代人都绕不开的问题——内卷与躺平。我们每天熬夜刷题怕落后，看到同学拼到凌晨却又觉得这条路没意义；想躺平却又害怕被甩在身后。今天我要告诉大家：在「内卷」与「躺平」之间，确实存在第三条路——它不是中庸，不是妥协，而是从「被动反应」走向「主动设计」。这条路的入口，藏在三个我们容易忽略的真相里。

📍 评分点：✓ 演讲格式（称呼 + 大家好） ✓ 共情开场（「我们都绕不开」+ 用毕业班场景） ✓ 明确立场 ✓ 预告 3 个分论点 ✓ 原创视角「被动反应 → 主动设计」

📜 范文 · 第 2 段（Body 1 / TEEL ~230 字）

T（主题句）：第三条路的第一个真相是——内卷与躺平共享同一个心理结构：「被动反应」。

E（Extend）：表面上，内卷者拼命加速、躺平者主动撤退，似乎是两种相反的选择。但仔细看，两者都不是在「设计自己想要的生活」——内卷者是被「不能输」的恐惧追赶，躺平者是被「赢不了」的疲惫击退。两者共享同一种心理姿态：让外在的规则替我决定如何生活。心理学家武志红一针见血地指出：「内卷与躺平不过是同一种焦虑的两副面孔。」

E（Evidence）：项飙教授把内卷形容为「一个陀螺被敲打」——转得很快却没有突破。躺平者把陀螺扔掉了，看似洒脱，其实仍然是被这套游戏规则定义的——他不是「不玩」，而是「输了不想再玩」。这就解释了一个奇怪的现象：很多躺平青年内心并不平静，他们仍然焦虑、仍然在比较——只是把比较从「赢过别人」换成了「确认自己已经退出」。

L（Link）：所以第三条路的起点，不是选择「卷」或「躺」——而是看清这两者其实是同一陷阱的两扇门。我们要先从「被动反应」的状态中醒来。

📍 评分点：✓ 原创视角（「同一心理结构」） ✓ 高级论据（武志红 / 项飙） ✓ 「陀螺被敲打」比喻（语言力量） ✓ Link 段过渡到 Body 2

📜 范文 · 第 3 段（Body 2 / TEEL ~230 字）

T（主题句）：第三条路的第二个真相是——主动设计的核心，不是「更努力」，而是「重新提问」。

E（Extend）：内卷者问的问题是「我怎么赢过他」，躺平者问的问题是「我何必跟他比」——但第三条路上的青年问的问题是：「什么对我有意义？」这个问题看似简单，其实极难——因为它要求我们暂时关闭外界的声音，问自己：在所有人的期待之外，我到底想成为什么样的人？北京大学心理咨询中心 2023 年的研究显示，越来越多的 90 后、00 后不再以「名校 / 大厂 / 高薪」作为人生坐标，而是开始构建自己的意义系统。

E（Evidence）：第三条路有三种典型形态。第一种是「专业内化」——程序员不再为升职卷代码量，而是为写出更优雅的算法专注。第二种是「价值重构」——东京、上海的极简主义青年群体快速增长，工资不高但生活精致，重视健康与深度社交。第三种是「使命驱动」——蚂蚁森林、乡村教育志愿者、独立纪录片人，他们的动力来自「我想让世界因我有所不同」，而不是「我要赢过谁」。这三种形态有一个共同点：他们的努力不再依赖于「比别人好」。

L（Link）：当我们把问题从「我怎么赢」换成「什么对我有意义」时，第三条路就从抽象变成了具体。

📍 评分点：✓ 原创视角（「重新提问」） ✓ 高级论据（北大研究 / 三种形态：专业内化 / 价值重构 / 使命驱动） ✓ 排比句式 ✓ 议题深度

📜 范文 · 第 4 段（Body 3 / TEEL ~230 字）

T（主题句）：第三个真相，是一个文学家在 60 年前就给我们的预言——当一个十七岁少年看不到第三条路时，他可能就走向了铁轨。

E（Extend）：白先勇在 1961 年写了一部小说——《寂寞的十七岁》。少年杨云峰，每天拼命背英文（这就是「内卷」的最早形态），背不出来就跑到新公园逃学（这就是「躺平」的最早形态）。他困在两条路之间，而第三条路从未在他面前打开。

E（Evidence）：你们也许会问，杨云峰生活在 60 年前的台北，跟我们有什么关系？关系大着呢。当父亲对他大喊「你还是个人哪？想做什么呀？」时，那些反问句不期待回答，只期待沉默——这跟今天父母问我们「你不努力将来怎么办？」其实是同一种语言形式。当同学把杨云峰最脆弱的告白信钉在黑板上群嘲时——这跟今天朋友圈截图嘲笑、群聊孤立同学，本质是同一种霸凌。当杨云峰最后趴到铁轨上时，白先勇用一个具体的少年告诉我们：当三重压力（学业 / 家庭 / 同伴）叠加而第三条路缺席时，悲剧就发生了。

L（Link）：所以第三条路不是抽象的哲学，它是救命的方向。今天的我们比杨云峰幸运的地方，不是压力变小了，而是我们已经知道——这条路存在，问题只是我们要不要走上去。

📍 评分点：✓ 原创视角（用文学文本作论据，避开陈词滥调） ✓ 议题深度（连接 60 年前 + 当下） ✓ 设问句（你们也许会问） ✓ 「以小见大」（杨云峰 = 当代青年）

📜 范文 · 第 5 段（结论 ~115 字）

亲爱的同学们：我知道，站在毕业班这个十字路口，我们每一个人都背着沉重的包袱——父母的期待、社会的标准、未知的未来。我也知道，「主动设计」听起来比「内卷」「躺平」都更难。但请你们记住：内卷让我们越拼越累，躺平让我们越退越空，唯有主动设计，能让我们越走越像自己。让我们不再做被敲打的陀螺，让我们不再做扔掉的陀螺，让我们做自己人生的雕塑师。谢谢大家！

📍 评分点：✓ 共情结尾（我知道你们的处境） ✓ 排比号召（「内卷让...躺平让...唯有主动设计...」/「让我们不再...让我们不再...让我们做...」） ✓ 演讲稿格式（谢谢大家） ✓ 重申立场
