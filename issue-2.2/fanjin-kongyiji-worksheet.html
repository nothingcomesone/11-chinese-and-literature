B ▎《范进中举》发疯段精读（Q3） Close reading

精读范围：《范进中举》(吴敬梓《儒林外史》第三回) 范进得知中举后当场发疯一段——从范进看榜后狂喜昏厥、拍手大笑、飞跑跌入水塘，到胡屠户一巴掌将他打醒、众邻居与张乡绅闻讯而来的全过程。

3 分(a) 范进得知自己中举后，当场"发疯"。请从选文中找出范进发疯时的 2-3 处具体表现，并逐字引用原文佐证（每处须带「」引号）。简要说明这些表现共同刻画出范进怎样的精神状态。

💡 提示

这一问考"识别+引文"。先快速圈出选文里范进"反常"的动作或语言（看榜后的反应→昏厥→大笑→奔跑→跌倒），挑 2-3 处。每一处都要从原文逐字抄下、加「」，不能改字、不能复述大意。最后用一句话概括他此刻的精神状态。注意：只描述"他做了什么"，还不需要分析手法——手法留到(b)。

5 分(b) 作者是用哪些方法来刻画范进"发疯"以及周围众人的反应的？请找出其中三种不同的手法，各引一处原文，并分别说明这种手法在此处产生了怎样的表达效果。

🛠️ 提示（TEE：手法 → 引文 → 效果，回扣"讽刺科举"）

题干故意不替你点明手法——请自己从这份术语清单里挑选：动作描写 / 语言描写 / 心理描写 / 细节描写 / 侧面描写 / 对比 / 反讽 / 夸张 / 白描。每一格按 TEE 三步填：① 手法（只用清单内术语） → ② 引文（从原文逐字抄，带「」，例如可考虑「噫！好了！我中了！」一句作示范） → ③ 效果——必须回扣"讽刺科举"这一题旨（说明这一笔如何揭示科举制度对人的扭曲、可笑或可悲）。三格手法不能重复；光给术语不解释=两张皮，不得分。

C ▎《孔乙己》手法精读 Techniques

【孔乙己·手法精读】——逐句细读鲁迅的文字。每行先读引文，再在右栏自己填出「手法 + 效果」：先判断作者用了哪一种手法，再想这样写让你读出了什么。

原文句子手法效果
「孔乙己是站着喝酒而穿长衫的唯一的人」
开篇第一句同时写了他喝酒的'姿态'和他穿的'衣着'。在咸亨酒店里，'站着喝酒'通常是哪一类客人？'穿长衫'又往往代表什么身份？作者把这两点写在同一个人身上，用了什么手法？再想想'唯一'这个词加在这里，让你对孔乙己产生了怎样的印象？请填：手法 + 效果。
「他对人说话，总是满口之乎者也」
'之乎者也'是古书里的文言词。一个在酒店里和普通客人打交道的人，开口却总是这些词。作者是借他的什么来刻画这个人？这样的说话方式让你看出孔乙己怎样的身份意识或心理？请填：手法 + 效果。
「便排出九文大钱……他从破衣袋里摸出四文大钱」
同一个人付钱，前一次'排出九文'，后一次'摸出四文'。先比较'排'和'摸'这两个动词，再比较'九文'和'四文'这两个数字——作者把前后两处放在一起，是什么手法？这一前一后，让你读出孔乙己的处境发生了怎样的变化？请填：手法 + 效果。
「窃书不能算偷……读书人的事，能算偷么？」
'窃'和'偷'意思相近，孔乙己却坚持要把它们分开。他这样为自己辩白，你读了觉得可笑还是可悲？作者写下这句话，对孔乙己、以及他所信奉的那一套读书人观念，流露出怎样的态度？这种'说的和实际相反'的写法是什么手法？请填：手法 + 效果。
「他便给他们茴香豆吃，一人一颗……不多不多！多乎哉？不多也」
作者没有直接评价孔乙己是什么样的人，而是写下'一人一颗'这个小动作和'多乎哉？不多也'这句口头禅。这种从极小处落笔来写人的写法是什么手法？这一颗豆子和这句文绉绉的话，分别让你看见孔乙己性格里的哪一面？请填：手法 + 效果。
「店内外充满了快活的空气」
孔乙己被人取笑、被人揭短，可作者偏偏去写'别人'的反应——'快活的空气'。作者不直接评说孔乙己的处境，而是写周围人的态度，这是什么手法？众人的'快活'和孔乙己当时的境遇之间，形成了怎样的落差？这落差让你读出作者怎样的用意？请填：手法 + 效果。
「大约孔乙己的确死了」
'大约'表示不确定，'的确'表示确定——两个意思相反的词却用在同一句里。这是作者写错了，还是有意这样写？'大约'透露出旁人对孔乙己的什么态度？把这两个词放在一起，又让你对孔乙己的结局产生怎样的感受？请填：手法 + 效果。

1. 聚焦动作之变：把'便排出九文大钱'和'他从破衣袋里摸出四文大钱'两处并排细读。'排'与'摸'一字之差，分别让你看出孔乙己怎样的心理与处境？钱数从九文到四文，这'动作 + 数字'的前后变化，又交代了他命运的什么转折？请结合这两处引文，自己分析作者这样写产生了怎样的表达效果。

2. 聚焦众人之笑：通读全文，找出写到'笑'或'快活'的句子（如'店内外充满了快活的空气'）。作者为什么反复去写旁人的反应，而不直接抒发对孔乙己的同情？请你判断：这是哪一种手法？众人的态度与孔乙己的处境形成了怎样的关系？这样写，对你理解当时社会、以及这篇小说要批判的东西，有什么作用？请结合引文写出你的分析。

E ▎Q4 跨文本比较写作 Comparative essay

✍️ 题目（约 350–450 字）

题目（Q4 跨文本比较·25 分）：「同一套科举/旧式教育，能把读书人捧成癫狂，也能把读书人逼成枯骨。」请比较《范进中举》中的范进与《孔乙己》中的孔乙己，分析两位作者如何通过内容与语言特色，揭示这套教育制度对读书人的扭曲与吞噬。（议题 2.2 教育） 学生任务：写一篇四段比较议论文（一部作品独立成一段）。开头点议题、亮立场、点两部作品；范进段、孔乙己段各用三个小 TEEL；结尾以「综上所述」收束于「殊途同归·制度之过」。本单只给支架与提示，请自行填写、不要照抄任何范文。

🔍 审题

  • 议题：本题考的是『科举/旧式教育制度如何扭曲、吞噬读书人』——焦点在『制度之过』，不是个人品德好坏，落笔前先想清楚『谁在害人』。
  • 立场：必须鲜明一边倒（如『制度是元凶』），切忌骑墙——不要写『一方面…另一方面…』或『制度也有好处』来稀释观点。
  • 范围：只能写范进与孔乙己两个人物、两部作品；一部独立成一段，比较点放在开头与结尾的『殊途同归』，正文段内不要混引另一部作品。

四段规划表

段落写作指引我的笔记
第①段·开头（50–80 字）三件事，写完即止，勿展开论据：①点议题——一句话点出『旧式教育/科举对读书人的扭曲』；②亮立场——用『我认为/不难发现……正是制度之过』表明一边倒态度；③点两部作品与两个人物——《范进中举》的范进、《孔乙己》的孔乙己。提示句式：『______，正是这套制度______的写照。』禁:开头超过 80 字、先讲故事背景。

第③段·孔乙己段（3 个小 TEEL，约 130–160 字）一部一段，只写孔乙己、只引《孔乙己》。与范进段换用不同手法以示对比。小T1【身份的夹缝】T:既非长衫主顾亦非短衣帮→E:旧教育给了他读书人的执念却给不了出路→自选一处原文（如『孔乙己是站着喝酒而穿长衫的唯一的人』『他对人说话，总是满口之乎者也』）→自己命名手法、说效果→L 扣『制度塑造畸形身份』。小T2【迂腐与自欺】T:偷书被讥仍以读书人自辩→E→自选一处原文（如『窃书不能算偷……读书人的事，能算偷么？』『君子固穷』『显出不屑置辩的神气』）→自己命名手法、说效果→L 扣『制度如何扭曲人的精神』。小T3【被吞噬的结局】T:被打折腿、众人取笑、悄然消失→E→自选一处原文（如『先写服辩，后来是打，打了大半夜，再打折了腿』『满手是泥，原来他便用这手走来的』『店内外充满了快活的空气』『大约孔乙己的确死了』）→自己命名手法、说效果→L 回扣议题:制度的吞噬与世态炎凉。提示:众人的『快活』与孔乙己的惨境形成强烈反差，可考虑用『对比』或『侧面描写』来分析。忌复述情节、忌只贴手法标签不说效果。
第④段·综上所述（50–80 字）以『综上所述』或『总而言之』开头。三件事：①收束比较——范进被『捧成癫狂』、孔乙己被『逼成枯骨』，一荣一枯，殊途同归；②点制度之过——指出二人命运相反却同为科举/旧式教育所害，揭示『制度之过』；③升华——一句简短有力的议题回扣（如『扭曲读书人的，从来不是个人，而是科举的牢笼』）。禁:补新论据、补新手法、超过 80 字。
